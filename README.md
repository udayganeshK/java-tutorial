# Java Spring Boot REST Tutorials

This repository contains tutorials and lab instructions for building RESTful APIs with Java Spring Boot, tailored for Angular developers.

## Structure
- `docs/` — Static site content for GitHub Pages (landing page, topics)
- `topics/` — Topic-specific markdown files (for future expansion)
- `tutorials/` — Example tutorials and advanced guides

## How to View the Site
1. Visit the [GitHub Pages site](https://udayganeshk.github.io/java-tutorial/) after deployment.
2. Or, browse the markdown files directly in this repository.

## How to Contribute
- Fork the repo, make changes, and submit a pull request.
